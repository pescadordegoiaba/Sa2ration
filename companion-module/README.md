# Sa2ration Companion

Módulo instalável em Magisk, KernelSU/SukiSU/ReSukiSU e APatch. Fornece serviço de boot, diagnóstico e protocolo seguro para adaptadores não lineares/vendor.

```shell
./gradlew packageCompanionModule
```

Saída: `build/outputs/module/Sa2ration-Companion-1.0.0.zip`. O módulo base não anuncia gama ou LUT como suportada sem adaptador validado para o dispositivo.
